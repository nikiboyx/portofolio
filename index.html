Siapa Saya?

Saya adalah mahasiswa Pendidikan Matematika di FKIP semester akhir. Saya memiliki ketertarikan dalam bidang pemrograman, khususnya dalam pengembangan website dengan menggunakan PHP, HTML, CSS, JavaScript, SQL database dan Python. Selain itu, saya juga aktif membuat konten edukasi dan motivasi di platform YouTube dan TikTok.

Motto hidup saya adalah "Hidup akan susah bila terlalu dipikirkan". Dengan semangat belajar yang tinggi, saya terus mengembangkan kemampuan di bidang teknologi dan pendidikan.
